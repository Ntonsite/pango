# Pango — Frontend

React 19 + Vite. See the [root README](../README.md) for the full project
(backend, Docker Compose, roles) and [TEST_USERS.md](../TEST_USERS.md) for
demo accounts.

## Local development

```bash
npm install
npm run dev      # dev server, expects the backend at /api (see vite.config.js proxy)
npm run lint
npm run build
```
